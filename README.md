# Retail UI (Nozdor)

![Client](https://img.shields.io/badge/Client-3%2E3%2E5a-blue) ![Version](https://img.shields.io/badge/Version-1%2E0-blue) ![License](https://img.shields.io/badge/License-MIT-green)

UI, вдохновлённый Retail-версией. Этот аддон перерисовывает существующий интерфейс.

## Возможности

* Панели действий
* Рамки юнитов
* Миникарта
* Полосы применения заклинаний
* Интерфейс транспорта
* Сетка для настройки расположения элементов интерфейса
* Здоровье цели с окрашиванием по классу
* Масштабирование интерфейса

## Быстрый старт

Введите `/rui edit` чтобы открыть сетку.
Введите `/rui default` чтобы сбросить настройки.
